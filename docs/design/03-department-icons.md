## 아이콘 시스템 규칙

### 기본 구조
- **크기**: 64×64px 기준 (SVG 벡터, 모든 사이즈 지원)
- **스타일**: 픽셀아트 (기존 캐릭터와 일관)
- **구성**: 캐릭터 + 직군 소품 + 부서 컬러 악센트
- **배경**: 투명 (어떤 배경에서도 사용 가능)
- **Soul spark**: 각 캐릭터 우상단에 부서 컬러 스파크

### 픽셀 그리드
- 8×8 메인 그리드 (6px per pixel at 64px)
- 머리: 3×2 블록 (시안/스킨톤)
- 눈: 1px 포인트 (부서별 컬러)
- 몸: 부서 컬러 기반
- 소품: 1~2 px 악센트

## 사용 컨텍스트

### 오피스 뷰
- 각 캐릭터가 부서별 자리에 배치
- 소품은 책상 위에 별도 렌더링 가능
- 부서 컬러로 영역 구분 (바닥 타일 / 벽 컬러)

### 에이전트 카드
- 32×32 아이콘으로 축소 사용
- 부서 컬러 배지와 함께 표시
- hover 시 64×64 확대

### Pricing 페이지
- 티어별로 캐릭터 수가 증가하는 일러스트
- Starter: CEO 1명 → Enterprise: 10명 전원

### 프로필/설정
- 128×128 확대 버전 (Soul 소개 페이지)
- Soul 편집 시 캐릭터 커스터마이징 미리보기
